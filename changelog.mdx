<Update label="2026-04-15" description="v1.2.0" tags={["feature","improvement"]}>
  ## New Features

  - Added automated email alerts for low inventory levels, preventing last-minute ingredient shortages.

  - Introduced batch inventory connection, allowing you to track ingredients through production runs and into live inventory for products.

  - Added Inventory tracking feature to the mobile app.

  ## Improvements

  - Enhanced order management dashboard to allow for many products in the list view.

  ## Bug Fixes

  - Fixed inventory tracking to use order created instead of order completed actions

  - Resolved the persistent error with additional decimals places added to inventory and recipe numbers.
</Update>

<Update label="2026-04-01" description="v1.1.0" tags={["feature","bugfix"]}>
  ## New Features

  - Added custom field functionality to inventory, product, batch and timesheets entries.

  - Added data import and export features for all entry types for faster setup and complete data export. making your data 100% yours to move and change as you wish.

  ## Improvements

  - Improved product margin calculator to allow for half recipes to full sellable units

  - Adjusted mobile views for orders and timesheet entries for easier view.

  ## Bug Fixes

  - Corrected customer list views not saving after logout.

  - Fixed import limit error failing with no meaningful message.
</Update>

<Update label="2026-03-25" description="v1.0.0" tags={["launch"]}>
  ## CASKA LIVE LAUNCH!

  - Caska app launched with full functionality for both web and native mobile apps.

  - iOS and Android beta apps available by invite
</Update>
