## Prerequisites

Before diving in, ensure you have the following ready:

<Callout kind="info">
- A business email address for account verification
- Basic details about your food manufacturing operation (e.g., primary ingredients, customer types)
- Access to your production schedule for initial setup
</Callout>

Caska works in any modern web browser. No downloads required.

## Create and Verify Your Account

Sign up takes under 2 minutes. Follow these steps to get started.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit [https://go.caska.app/signup](https://go.caska.app/signup) and create your account.

    Enter your business email, company name (e.g., "FreshBake Foods"), and a secure password.

    <Callout kind="tip">
      Use your production manager's email for primary access.
    </Callout>
  </Step>

  <Step title="Verify Email" icon="mail">
    Check your inbox for the verification email from Caska.

    Click the link to activate your account. If it doesn't arrive in 5 minutes, check spam.
  </Step>

  <Step title="Complete Onboarding" icon="check-circle">
    Log in at [https://go.caska.app](https://go.caska.app).

    Answer the quick onboarding questions:
    - Business type: Food manufacturer
    - Key focus: Inventory and production
    - Team size: e.g., 5-20 employees
  </Step>
</Steps>

## Set Up Your Dashboard

After onboarding, customize your dashboard for food operations.

<Steps>
  <Step title="Connect Basic Info" icon="settings">
    From the dashboard, go to **Settings > Company Profile**.

    Add your warehouse address and default units (e.g., kg for ingredients, cases for finished goods).
  </Step>

  <Step title="Add First Inventory Items" icon="package">
    Navigate to **Inventory > Add Item**.

    Create your first item:

    ```
    Name: All-Purpose Flour
    SKU: FLOUR-001
    Unit: kg
    Reorder Level: 50 kg
    Current Stock: 200 kg
    ```

    Repeat for essentials like sugar or yeast.
  </Step>
</Steps>

## Explore Core Modules

Switch between modules using the sidebar. Here's a quick tour.

<Tabs>
  <Tab title="Inventory" icon="package">
    Track stock levels in real-time. Set reorder alerts to avoid shortages.

    View your flour item and simulate a usage: deduct `10 kg` for a test batch.
  </Tab>

  <Tab title="Production" icon="factory">
    Plan your first batch.

    Create a new production run:
    - Recipe: Bread Batch #1
    - Ingredients: Flour `{200 kg}`, Yeast `{5 kg}`

  <Tab title="Orders" icon="shopping-cart">
    Add a test customer order.

    New Order:
    - Customer: Local Grocery
    - Items: Bread `{500 loaves}`

  </Tab>
</Tabs>

## API Quickstart (Optional)

Integrate Caska with your existing tools via API.

<CodeGroup tabs="JavaScript,cURL">
  ```javascript
  const response = await fetch('https://api.example.com/v1/inventory', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      name: 'All-Purpose Flour',
      sku: 'FLOUR-001',
      unit: 'kg',
      stock: 200
    })
  });
  ```
  ```bash
  curl -X POST https://api.example.com/v1/inventory \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "name": "All-Purpose Flour",
      "sku": "FLOUR-001",
      "unit": "kg",
      "stock": 200
    }'
  ```
</CodeGroup>

<ParamField path="Authorization" param-type="string" required="true">
  Bearer token obtained from dashboard settings.
</ParamField>
